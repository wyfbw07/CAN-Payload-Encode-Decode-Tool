# CAN_DBC_DECODE_TOOL

## Usage

argv[1]:  DBC file address

argv[2]: Message ID in DEC

argv[3]: Payload in HEX

argv[4]: Signal Name

## Sample Usage

### Sample usage 1:

argv[1]: /Users/filelocation/XVehicle.dbc

### Sample usage 2:

argv[1]: /Users/filelocation/XVehicle.dbc

argv[2]: 336

argv[3]: D0,87,F0

### Sample usage 3:

argv[1]: /Users/filelocation/XVehicle.dbc

argv[2]: 336

argv[3]: D0,87,F0

argv[4]: EngineTemp
